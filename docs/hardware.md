# Hardware Utilizado
## Componentes:
- ESP32 DevKit V1
- Sensor DS18B20
- Potenciômetro simulando microfone KY-038
- Buzzer piezoelétrico
- Resistor 4.7k
- Protoboard e jumpers

## Ligações:
- DS18B20 DATA → GPIO4
- DS18B20 VCC → 3.3V
- DS18B20 GND → GND
- Resistor 4.7k entre DATA e VCC
- Potenciômetro SIG → GPIO34
- Buzzer → GPIO5
